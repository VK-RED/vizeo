# Vizeo

Vizeo is an open-source project designed to simplify video transcoding, subtitle generation, and thumbnail creation through a single API endpoint. It aims to reduce the manual efforts involved in Video Creation.

## Features (For Now)

- **Video Transcoding**: Easily convert videos into multiple formats compatible with various platforms.
- **Subtitle Generation**: Automatically generate subtitles for your videos.
- **Thumbnail Creation**: Create thumbnails for videos to enhance user engagement.
  
**If you feel any new features can be added. Please feel free to raise an issue !!!**
